## 三种使用方式

### 方式 A：纯图形界面（最常见）

```
安装 → 设置行情引擎目录 → 服务页启动 → 交易/行情页使用
```

适合：日常查仓、手动下单、管理交易信号。

### 方式 B：AI 助手（Cursor / Codex 等）

```
启动网关 → 配置 MCP → 对话查行情/持仓
```

适合：AI 玩股、智能盯盘、自然语言查数据。详见 [AI 接入指南](./CURSOR.md)、[AI 交易示例](./AI交易示例.md)。

### 方式 C：本机命令行（CLI）

```
启动网关 → PowerShell 调用 aitradegw.exe
```

适合：脚本自动化、计划任务、本机调试。详见 [CLI 使用说明](./CLI使用说明.md)。

### 方式 D：交易信号自动化

```
启动网关 → 交易信号页安装组件 → 刷新并发送
```

适合：策略入队后自动报单，减少手点确认。

## 与 QMT 有什么不同

| | 传统 QMT 路线 | 通用 AI 交易网关 |
|---|--------------|-----------------|
| 门槛 | 向券商申请、可能付费 | **本地安装即可** |
| 使用方式 | 偏编程、偏机构 | **GUI + AI 双通道** |
| AI 接入 | 需自行对接 | **内置 MCP，Cursor / Codex 直连** |
| 安全 | 视券商规则 | **默认模拟，实盘双重确认** |
