## 页面一览

| 页面 | 你能做什么 |
|------|------------|
| **服务** | 一键启动/停止网关与 AI 接口；查看运行是否正常 |
| **初始化** | 检测 Python 与量化环境，缺什么一键补 |
| **交易** | 查资金、持仓、委托；模拟或实盘买卖撤 |
| **交易信号** | 查看待发送信号、一键发送、下单后自动跟发 |
| **行情** | 查个股快照、五档、简要资金信息 |
| **CLI** | 内置快捷测试，验证网关是否连通 |
| **设置** | 选择行情引擎目录、是否允许实盘等 |
| **日志** | 查看运行记录，排查问题 |
| **激活** | 输入注册码在线激活、检查更新 |
| **AI 连接器** | 安装 / 检测 WorkBuddy 等本机 MCP 连接器 |
| **计划任务** | 盘后日线 / 财务汇总包下载；交易日定时更新；查看远端与本地更新时间 |
| **声明** | 免责声明与在线文档 |

### 计划任务页（盘后数据）

1. **设置** 页确认「安装目录」指向通达信行情引擎根目录（其下应有 `vipdoc`）
2. 打开 **计划任务**：可「刷新状态」对比远端 / 本地更新时间
3. **立即下载**：日线（`hsjday.zip`）、财务（`tdxfin.zip`）等；远端未变会自动跳过，避免重复解压
4. **启用定时**：如「启用日线定时」后，需保持 **服务** 页网关运行；仅在 **A 股交易日** 到点执行
5. AI / 脚本：MCP 工具 `tdx_vipdoc_status`、`tdx_vipdoc_download`、`tdx_schedule_list` 等；CLI `aitradegw schedule vipdoc-status` / `vipdoc-download`

## 服务页 — 从这里开始

![服务页](./IMG/1.png)

打开软件后，先到 **服务** 页：

1. 看四行状态（网关、AI 接口、行情核心、交易客户端）
   - **绿色** = 正常 · **橙色** = 异常 · **灰色** = 未启动
2. 点击 **启动网关** — 这是使用一切功能的前提
3. 若要用 Cursor / Codex 等 AI 工具，再点 **启动 MCP**
4. 需要交易时，点 **启动交易** 并在 **交易客户端** 登录
5. 若 **先开网关、后开行情/交易客户端**，点 **程序重新初始化** 重新连接 TQ（不必重启网关）
6. **服务自愈**：保持控制台常驻时，网关 / MCP 异常退出会自动拉起（日志可见「自愈：…已自动重启」）；点「停止」则关闭自动拉起。行情主程序 `TdxW.exe` 重新运行后，网关也会自动重连 TQ

下方还有 **健康检查**、**连接状态**、**行情探针**，不确定是否连通时点一下即可。

纯 GUI + AI 用户可跳过此页。

## 交易页 — 查仓与下单

![交易页](./IMG/3.png)

**使用前**：服务页已启动网关，且交易客户端已登录。

- 选择账户 → 查 **资金**、**持仓**、**委托**
- **买入 / 卖出 / 撤单**：默认 **模拟**，不会真下单
- 要实盘：先在 **设置** 打开「允许实盘」，下单时再勾选确认

---

## 交易信号页 — 策略信号不再手点

![交易信号页](./IMG/4.png)

量化策略下单后，部分委托会进入 **交易信号队列**，需手动确认。本页帮你 **自动同步、一键发送**：

1. 服务页 **启动网关**
2. 点 **安装组件**（首次使用，一键部署所需页面）
3. **刷新列表** → 查看待发送信号
4. **全部发送** 或选中后发送；可开启 **自动轮询** 与 **下单后跟发**

## 行情页 — 快速看个股

![行情页](./IMG/5.png)

输入证券代码，即可查看：

- 实时快照与五档
- 简要资金/持仓摘要（需已登录交易端）

适合日常盯盘、核对数据，无需反复切换多个窗口。

## CLI 页 — 快速验连通

![CLI 页](./IMG/6.png)

内置常用测试命令与快捷按钮。网关启动后，在此输入命令或点快捷测试，即可看到 JSON 结果。

适合进阶用户或客服远程协助排查；普通用户用 **服务** 页的健康检查即可。

完整命令说明见 **[CLI 使用说明](./CLI使用说明.md)**；AI 对话示例见 **[AI 交易示例](./AI交易示例.md)**。

## 设置页

| 项 | 说明 |
|----|------|
| **行情引擎目录** | 本地行情软件安装根目录（安装程序通常已自动填写） |
| **允许实盘** | 关闭时只能模拟；开启后仍须在每笔交易确认 |
| **监听地址 / 端口** | 一般保持默认；仅局域网共享时联系作者指导 |

配置保存在安装目录 `bin\settings.json`。

## 激活页

- 填写 **注册码**、**备注名** → **在线激活**（需联网，仅首次）
- 激活后可 **校验授权**、**检查更新**
- 换电脑需联系客服 **解除绑定** 后再激活

## 小提示

- **关闭窗口** 会最小化到系统托盘，网关可继续在后台运行；托盘图标可重新打开
- 升级软件时退出 GUI 与网关，覆盖安装即可，**一般不必重新激活**
- 遇到问题：先看 **日志** 页，或对照 [安装指南](./USER-INSTALL.md) 检查清单

## 常见问题

| 现象 | 怎么办 |
|------|--------|
| 网关一直是灰色 | **设置** 里确认 **行情引擎目录** 正确 → 点 **启动网关** |
| 交易查不到数据 | 服务页 **启动交易** 并在交易客户端登录 |
| 交易信号装不上 | 先启动网关；关闭行情客户端后重试 **安装组件** |
| 激活失败 | 检查网络；确认注册码未绑其它电脑 |
