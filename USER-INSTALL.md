# 安装与使用指南

面向 **正式用户**：下载安装包或绿色包即可使用，无需编程环境。

---

## 系统要求

- Windows 10 / 11（64 位）
- 本机已安装 **通用行情引擎**（含量化模块，安装时选择对应目录即可）
- 需要实盘交易时：**交易客户端** 能正常登录
- 首次 **在线激活** 需联网

---

## 方式一：安装程序（推荐）

1. 运行 **`AITradeGW_Setup_*.exe`**
2. 安装过程中 **选择行情引擎目录**（本地行情软件的安装文件夹）
3. 安装完成，勾选 **启动图形控制台**（或从开始菜单打开「通用AI交易网关」）
4. 在 **服务** 页点击 **启动网关**
5. 在 **激活** 页输入注册码 **在线激活**（首次使用）

> 安装前建议关闭交易客户端及正在运行的旧版本网关。

---

## 方式二：绿色包

1. 解压到固定目录，例如 `C:\Program Files\AITradeGW\`
2. 运行 `bin\aitradegw-gui.exe`
3. **设置** → 填写 **行情引擎目录** → **保存**
4. **服务** → **启动网关**
5. **激活** → 在线激活

目录结构示意：

```text
AITradeGW\
  bin\
    aitradegw-gui.exe      ← 图形控制台（日常入口）
    aitradegw-gateway.exe  ← 网关（服务页自动管理）
    aitradegw-mcp.exe      ← AI 接口（服务页自动管理）
    settings.json          ← 配置文件
  QUICKSTART.txt           ← 快速说明
```

---

## 首次使用检查清单

- [ ] **行情引擎目录** 已在 **设置** 中正确填写
- [ ] **服务** 页网关状态为 **绿色**
- [ ] 点 **健康检查** 显示正常
- [ ] 需要交易：已启动并登录 **交易客户端**
- [ ] 需要交易信号：**交易信号** 页 → **安装组件** → **刷新列表**
- [ ] 高级版：已在 **激活** 页完成在线激活

---

## 连接 AI（Cursor / Codex 等）

网关启动后，按 [AI 接入指南](./CURSOR.md) 配置 MCP，即可让 AI 查行情、查持仓。

也可在 **服务** 页一键 **启动 MCP**。

---

## 升级

1. 退出图形控制台、网关（服务页 **停止**）
2. 运行新版安装程序覆盖安装，或替换 `bin\` 下程序文件
3. **不必重新激活**（除非换电脑或客服解除绑定）
4. 交易信号功能升级后，可在 **交易信号** 页再次 **安装组件**

---

## 卸载

- 系统 **设置 → 应用** 中卸载「通用AI交易网关」
- 可选：删除 `%APPDATA%\aitradegw\`（含授权文件，卸载后换机需重新激活）
